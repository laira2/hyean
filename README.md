#미술품 판매 사이트 혜안

```mermaid
  gantt
    title A Gantt Diagram
    dateFormat  YYYY-MM-DD
    section 기획/디자인
    기획           :a1, 2024-06-03, 3d
    디자인     :after a1  , 14d
    section 기술개발
    메인 페이지      :a2, 2024-06-07  , 2d
    검색 기능       :after a2, 3d
    회원가입/로그인 :a3, after a2, 3d
    상세페이지/장바구니 :a4, after a3, 2d
    상품주문/결제 :a5, after a4, 3d
    추가 기능 개발 : after a5, 3d
    section 문서 작성
    발표 준비 : 2024-06-20, 2d
```
